# Two Leaves & A Bud – Responsive Tea Brand Landing Page

A modern, fully responsive tea ecommerce landing page built using pure HTML and CSS.  
This project focuses on premium UI design, responsive layouts, reusable sections, and smooth user experience without using any frontend framework.

---

# Features

## Responsive Design
- Fully responsive on all screen sizes
- Optimized for Desktop, Tablet, and Mobile devices
- Mobile-first responsive media queries

## Navigation System
- Sticky Navbar
- Hamburger Menu for mobile
- Slide-in Mobile Drawer
- Responsive navigation layout

## Hero Section
- Fullscreen banner
- Background image support
- CTA button
- Smooth visual hierarchy

## Product Showcase
- Best Seller horizontal carousel
- Product cards with:
  - Ratings
  - Tags
  - Add buttons
  - Hover effects

## Reviews Section
- Horizontal scrollable review cards
- Mobile swipe support
- Responsive card sizing

## Discover Section
- Search input UI
- Tea category vibe tags
- Interactive hover effects

## Journal / Blog Section
- Article cards
- Hover animations
- Responsive layouts

## Cafe & Wholesale Section
- Split layout design
- CTA buttons

## Footer
- Multi-column footer
- Newsletter signup
- Social media icons
